# Skylite

We're a flight simulation network based in China, working to make the FSD ecosystem more open and accessible, so any network can have the same tooling and features that VATSIM has, without the proprietary lock-in.

## Projects

### Open source

- [**neoswift**](https://github.com/skylitefly/neoswift): Fork of swift pilot client with full per-network configurability, fsd-autodiscovery, and VATSIM proprietary components removed
- [**fsd-autodiscovery**](https://github.com/skylitefly/fsd-autodiscovery): Open standard for automated FSD network configuration.
- [**pyfsd**](https://github.com/skylitefly/pyfsd): Pluggable Python asyncio implementation of the FSD server

### Coming soon

- **eaip-parser**: Parses China eAIP offline packages into a structured format for use by other services
- **afv-server**: Open source server implementation of AFV protocol
